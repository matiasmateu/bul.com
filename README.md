## Usage


#### 1. Clone and Install


git clone https://github.com/matiasmateu/bul.com.git

#### 2. Install dependencies ( In both backend and frontend )
```bash
npm install
```

#### 2.1. Run the server

```bash
nodemon
```
#### 2.2. Run the client

```bash
npm run start
```
